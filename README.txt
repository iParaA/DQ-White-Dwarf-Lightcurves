# DQ-White-Dwarf-Lightcurves
The data has been aquired from the TESS satellite and processed then plotted using Lightkurve.

lc = lightcurve
pg/pd = periodogram (pg is proper notation)
Each folder specifies the WD name, the 0,1,2, at the beginning of image file name is to order the files, followed by WD name, sector, and period.
